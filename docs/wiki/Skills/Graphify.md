# Skills/Graphify

Turn a folder of files into a knowledge graph (HTML, JSON, Obsidian export, GraphRAG-style queries). Trigger: `/graphify`.

**Canonical instructions:** [`skills/graphify/SKILL.md`](https://github.com/ldom1/ai-dotfiles/blob/main/skills/graphify/SKILL.md)

**Upstream:** [graphify.net](https://graphify.net/)

**Optional local uv clone:** set `GRAPHIFY_PROJECT` or `~/ai-dotfiles/config/graphify.env` (see repo `config/graphify.env.example`).
